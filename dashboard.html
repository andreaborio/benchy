Get started

No runs yet — three steps to your first benchmark score. This panel disappears once a run lands in results/runs.jsonl.

  1. Fetch a benchmark — open Benchmarks and download a set into data/ (pinned & hash-locked).
  2. Point or start your OpenAI-compatible server — use Start server in the Model server card, or launch your own; it is auto-detected on .
  3. Run N=25 — pick the benchmark in Run a test, keep N=25, tag it (e.g. baseline) and hit Run.

Thinking vs non-thinking

Runs explorer click a run to inspect every question

Live stream, accuracy & LLM activity update ~1.5s · tables 6s · live metrics & benchmark numbers come from results/ & server.log; the model id & host specs are auto-detected, reference baselines come from your config.json (Setup).
Methodology & caveats in README.md — small-N Wilson CIs, page-cache/thermal noise, non-thinking letter bias, decode t/s ±~10%. Rubric-graded benchmarks (e.g. HealthBench) score 0–100, not % correct, and are excluded from the MCQ macro-average.
